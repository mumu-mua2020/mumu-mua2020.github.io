第三周，吴恩达课后练习，日常报错

在做吴恩达深度学习课程第一课第三周的课后作业时，运行如下代码：

错误1：

错误代码：

​ plt.scatter(X[0, :], X[1, :], c=Y, s=40, cmap=plt.cm.Spectral)

错误原因：ValueError: 'c' argument has 1 elements, which is not acceptable for use with 'x' with size 400, 'y' with size 400.

其实就就是版本的原因啦！

改正：

import operator from functools import reduce plt.scatter(X[0,:], X[1,:], c = reduce(operator.add,Y), s = 40, cmap = plt.cm.Spectral);

错误2：

错误代码：

错误原因：ValueError: shapes (4,4) and (2,3) not aligned: 4 (dim 1) != 2 (dim 0)

其实就是维度，不一样，因为弄错了，x 和 y肯定会报错的。所以，改过来就好了

改正：

return (n_y,n_x,n_h) 调整为 正确的顺序即可， ==return (n_x,n_h,n_y)==
