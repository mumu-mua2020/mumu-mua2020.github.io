第三周，吴恩达课后练习，日常报错

在做吴恩达深度学习课程第一课第三周的课后作业时，运行如下代码：

错误1：

错误代码：

​ plt.scatter(X[0, :], X[1, :], c=Y, s=40, cmap=plt.cm.Spectral)

错误原因：ValueError: 'c' argument has 1 elements, which is not acceptable for use with 'x' with size 400, 'y' with size 400.

其实就就是版本的原因啦！

改正：

import operator from functools import reduce plt.scatter(X[0,:], X[1,:], c = reduce(operator.add,Y), s = 40, cmap = plt.cm.Spectral);

错误2：

错误代码：

错误原因：ValueError: shapes (4,4) and (2,3) not aligned: 4 (dim 1) != 2 (dim 0)

其实就是维度，不一样，因为弄错了，x 和 y肯定会报错的。所以，改过来就好了

改正：

return (n_y,n_x,n_h) 调整为 正确的顺序即可， ==return (n_x,n_h,n_y)==

HFETBG 基于层次适应度的进化算法测试问题基准生成器

基本流程：

输入:

  1. N个算法，$SA={A1,A2,A3……AN}$；
  2. 可调的基准实例生成器；
  3. 演化问题的进化算法（EPAI）:这里用的是==差分进化算法（DE）==

输出：

  1. 我们所需要的测试问题实例。（==基于HFETBG的一个套件集，有N个问题实例，每一个EA，每个问题都有他们唯一有利的问题==）

符号表示：

N：表示的是N个已经建立好了的进化算法（EAS）;

SA：SA={A1,A2，……{AN} 表示N个算法的集合；

Ai ：表示的是每一个算法；

AT: 表示的是Ai中最优的算法；

BA= SA - AT= {B1,B2,B3……BN-1} 被AT打败后的所有算法；

AM : 是BA中性能最优的算法；

模式识别之–线性分类器–Fisher（LDA）、最小平方五误差（SME）、感知机分类

这几天模式识别课布置了一个实验作业，就是用这用着三个分类器，对数据进行分类。

线性模型

什么是线性分类？我自己的理解就是，用一条线进行分类，用一条线，把两类进行分开，这就是线性分类。（当然，仅仅是在一维空间里面；假如在二维空间里面，就是一个面了；三维……一直到n维……）

线性模型的基本形式，
$$
f(x)=w*x+b
$$

$$
x =(x^1,x^2,x^3,……x^n)^T, 其中x^n是第n个样本的特征，
$$

$$
w =(w^1,w^2,w^3,……w^n)^T,其中，w是每个特征对应生成的权重。
$$
